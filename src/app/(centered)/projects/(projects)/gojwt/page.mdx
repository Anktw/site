When I was learning Go, I wanted to build a simple authentication service that uses JWT for user authentication. This project is a simple implementation of JWT-based authentication service in Go.

## Features
- User registration
- User login
- Token validation
- Middleware for protected routes
- Refresh token support
- Password hashing
